Simple Village Box


> 基于pixel的VillageBox 二次开发，已经取得他的授权。（竟然是同事……）代码其实没怎么变动，但主要调整了交易体系。更加适合生存玩家


### 主要修改点

+ 金币可以用金锭兑换，物品可以卖出得到银币或者铜币
+ 村民的名字不再是随机生成了，直接使用了职业作为名字
+ 删除了原来mod里面的食品，和一些职业
+ 通过交易和做任务你几乎可以获取所有原材料了（刷怪蛋和附魔书无法获取）
+ 新增末影商人，你可以买到一些珍贵物品了
+ 村民任务刷新时间更加频繁了
+ 村民死亡后重生时间更短了
