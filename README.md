Simple Village Box


> 基于pixel的VillageBox 二次开发，已经取得他的授权。（竟然是同事……）代码其实没怎么变动，但主要调整了交易体系。更加适合生存玩家


### 主要修改点

+ 金币可以制造了，去银行家那里用金条兑换
+ 你不能从矿工那里买到金条了
+ 所有物品价格上涨了，毕竟金币可以兑换出来了
+ 除了买物品，还可以卖物品了
+ 银币可以直接买到只有铜币能买到的物品了，但是价格会比铜币买的贵
+ 去掉了很多的食品item
